## 関数
```python
import pandas as pd
pd.resample(rule)
```
期間ごとの集計を行なう。  
###### 引数
#### rule
第１引数
- D: 毎日
- B: 毎営業日（月曜 - 金曜）
- W: 毎週（日曜始まり）
- M: 月末ごと
- SM: 15日と月末ごと
- Q: 四半期末ごと
- AまたはY: 年末ごと  
etc...

#### label
インデックス列のラベルを開始日時にするか終了日時にするか指定する。
- label='left': 開始日時
- label='right':終了日時

がラベルとなる。

#### closed
開始日時と終了日時のどちらを期間に含むかを指定する。
- closed='left': 開始日時 <= 期間 < 終了日時
- closed='right': 開始日時 < 期間 <= 終了日時

となる。

#### on
これまでの例のようにインデックス列が日時データであればそのままで問題ないが、インデックスではない列に日時データが格納されている場合、
引数onに日時データが格納された列名を指定するとリサンプリングが可能。

## 分布
### べき分布
０付近に多くの Sample数が集中し、それ以降、急激に減少していく分布。
Business の現場では非常に多く見られる。「パレートの法則」や「80:20の法則」などと呼ばれる Business の現場で多く見られる法則を裏付ける分布。
個々人の能力そのものというよりも、これまで積みあがった実績や貯蓄額が相互に影響しあうものに多く、一部の人が全体のうちの大多数を独占する、といった傾向をもつものに多く見られる
### 一様分布
「サイコロの目の頻度」や「Roulette のそれぞれの数字の頻度」など、何の偏りもない「平等」な確率によって成り立つ分布。
### 正規分布
自然界において頻繁に見られる重要な分布。
個々人が持つ身体能力などがそのまま反映されるようなものに頻繁に見られる
