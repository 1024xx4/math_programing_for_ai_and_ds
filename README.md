## 第２部　数理最適化 編

**「最適化問題」** にはいくつかの Pattern があるが **「内定化」** する方法と、それを解く流れについて理解してしまいさえすれば、あとは目の前
の問題がどの Pattern として解けるか考えるだけになる。

### 数理最適化が役立つ Sean

- これまで人手で行なっていたノウハウなどを自動化、より効率化できる方法を探りたい場面。
- ある指標を定め、その指標を最大もしくは最小にしたい場合に「最適化問題」として「内定化」する。

### 内定化

基本的には数式を用いるが言葉を記述して Programing に落とし込む、とうことを行なうこもと可能。
